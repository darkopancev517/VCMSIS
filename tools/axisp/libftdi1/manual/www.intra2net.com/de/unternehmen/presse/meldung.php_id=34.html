Intra2net AG erweitert Groupware um Push-Funktion

Intranator Business Server 6.1 bietet Datensynchronisation via ActiveSync für iOS, Android und Windows Phone

Tübingen, 28.10.2013 - Die Intra2net AG (www.intra2net.com), Anbieter für Security- und Groupware-Lösungen, stellt den Intranator Business Server 6.1 vor. Die neue Version des Groupware-Servers unterstützt jetzt die drahtlose Datensynchronisation für mobile Endgeräte via ActiveSync. Damit können Nutzer sämtliche Daten wie E-Mails, Kalender und Kontakte automatisch zwischen Intranator Business Server und ihrem Smartphone synchronisieren. Gemeinsame Ressourcen stehen so jederzeit plattform- und standortunabhängig zur Verfügung.

Die eingesetzte Push-Technologie basiert auf einer Open-Source-Implementierung von ActiveSync 14.1, dem De-facto-Standard zur Synchronisation von mobilen Endgeräten. Der Datenabgleich erfolgt automatisch via Push-Funktion. Hierfür muss auf dem Endgerät keine Zusatzsoftware installiert werden. Es lässt sich jedes moderne Smartphone oder Tablet einbinden, das auch mit Exchange genutzt werden kann. Dazu zählen alle iOS-, Android- und Windows Phone- basierten Geräte.

Als weitere Neuerung bietet Intranator Business Server 6.1 einen verbesserten Groupware Client für Microsoft Outlook mit voller Unterstützung von Outlook 2013 und einer deutlich komfortableren Konfiguration. Innerhalb eines E-Mail-Profils können jetzt mehrere Konten von unterschiedlichen Servern über separate Datendateien verwaltet werden. Zusätzlich werden Outlook-Kontaktgruppen automatisch synchronisiert.

Über Intranator Business Server
Der Intranator Business Server deckt als Komplettpaket typische Einsatzszenarien rund um Internet-Security und E-Mail ab. Die Linux- basierte Lösung ist für den Einsatz in Unternehmen mit fünf bis 50 Anwendern optimiert und wahlweise als Software-, Hardware- oder virtuelle Appliance erhältlich. Der Zugriff auf die Groupware erfolgt mit Outlook 2003 bis 2013, Web-Client und Smartphone. Angesichts der Abkündigung des verbreiteten Small Business Servers positioniert sich der Intranator Business Server 6.1 als eine interessante Lösung für Anwender, die nach einer preislich attraktiven Exchange-Alternative suchen.

Systemvoraussetzungen für ActiveSync
Unterstützt werden alle iOS-, Android- und Windows Phone-basierten Smartphones und Tablets sowie Microsoft Outlook 2013.

Preise und Verfügbarkeit
Intranator Business Server 6.1 steht für bestehende Kunden ab sofort per Update zur Verfügung. Neukunden können die Lösung über die rund 200 Fachhandelspartner der Intra2net AG beziehen. Eine Fünf-Benutzer-Lizenz kostet inklusive Updates für zwölf Monate 390,- Euro zzgl. Mehrwertsteuer.

Eine kostenlose 30-Tage-Testversion kann unter www.intra2net.com/de/download heruntergeladen werden. Weitere Informationen, die aktuelle Preisliste sowie ein Produktvergleich sind unter http://www.intra2net.com/de/produkte/intranator.php zu finden.
