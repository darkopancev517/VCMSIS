Software Version 6.1.8

Neue Version des HTTP-Proxies (14.07.2014)

  • Schnellere Webzugriffe durch verbesserte Architektur
  • Vollständige HTTP/1.1 Unterstützung
  • Fehlermeldungen je nach Browser-Sprache
Weitere Verbesserungen
  • Uhrzeiten in Firewall-Regeln werden jetzt automatisch an Sommerzeit / Winterzeit angepasst
  • Lokale Zeitzone unter "Dienste > Zeitabgleich" einstellbar
  • Unterstützung des "changeip.com" DynDNS-Dienstes
  • Verbesserung der Sicherheit am Linux-Basissystem (elfutils-0.158: CVE-2014-0172, grep-2.18: CVE-2012-5667, kernel-3.4.96: CVE-2014-4508, libtiff-4.0.3: CVE-2013-4243, lzo-2.0.8: CVE-2014-4607, nspr-4.10.6, nss-3.16.1, nss-softokn-3.16.1, nss-util-3.16.1, python-2.6: CVE-2014-1912, python3-3.3.2: CVE-2014-4616)
  • Aktualisierung von Linux-Basisdiensten (c-ares-1.10.0, grep-2.18, libtiff-4.0.3, squid-3.4.5, tzdata-2014e)
