Software Version 5.4.0

Intranator Security Gateway sowie globale Spam-Quarantäne (28.11.2011)


Unterstützung für Intranator Security Gateway
  • Umfassende Netzwerk-, Web- und Mail-Sicherheit am Gateway
  • Perfektes Zusammenspiel mit bestehenden Mail-Server
  • Volumenstaffel-Lizenzierung ab 5 Benutzer
Neue Funktion: Globale Spam-Quarantäne
  • Zentrale Quarantäne für Spam und Spamverdacht E-Mails
  • Zugriff unter "Dienste > E-Mailfilter > Quarantäne > Spam"
  • Tägliche Reports für den Administrator oder pro Benutzer
Optionaler Zugriff auf E-Mail Quarantäne mit Datenschutz-Passwort
  • Verbessertes Bandbreitenmanagement für Leitungen mit geringer Latenz
  • iPhone VPN: Begrüßungstext abschaltbar
  • Zugriff auf die E-Mail Warteschlange wieder möglich
  • Verbesserung der Sicherheit am Linux-Basissystem (bind-9.7.4: CVE-2011-4313, dbus-1.2.28: CVE-2011-2533, freetype-2.4.7: CVE-2011-3256, gnupg2-2.0.18: CVE-2010-2547, perl: CVE-2011-3597)
  • Aktualisierung von Linux-Basisdiensten (file-5.09, kernel-3.0.9, libassuan-2.0.1, libgpg-error-1.10, samba-3.5.12)
