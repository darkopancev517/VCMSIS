Software Version 5.1.4

Neuer Virenscanner sowie zahlreiche Aktualisierungen (07.05.2009)

  • Wichtiges Sicherheitsupdate am Virenscanner (FSC-2009-1)
  • Neuer F-Secure Virenscanner (fssp-7.02.73807)
  • Aktualisiertes VPN-System
  • Vereinfachung der VPN-Startmodi: "Wenn online" mit "Immer" verschmolzen
  • Neuer Linux Kernel mit aktualisierten Treibern
  • Statische NAT schreibt vom Zielrechner abgehende Verbindungen um
  • Aktualisierter POP3-Abholungsdienst
  • Proxyserver akzeptiert "Expect"-Anforderungen
  • Generierung von Firewallregeln in Randfällen verbessert
  • Verbesserung der Sicherheit am Linux-Basissystem (fetchmail-6.3.9, ghostscript-8.63: CVE-2008-6679, CVE-2009-0196, CVE-2009-0583 und CVE-2009-0792)
  • Aktualisierung von Linux-Basisdiensten (kernel-2.6.29.2, mdadm-2.6.9, openssh-5.2p1, procps-3.2.7, reiserfsprogs-3.6.21, strongswan-4.14, sqlite-3.6.12, tar-1.22)
