Software Version 5.3.1

Neue Proxy-Zugriffslisten sowie Fax-Seitenrand auf A4 beschränkt (11.05.2010)

  • Neue Proxy-Zugriffslisten: Chat und (Browser-)Spiele
  • Gesteigerte Erkennungsrate der bayesischen Spam-Datenbank
  • Fax-Seitenrand auf A4 beschränkt (keine "leere" Seite mehr)
Weitere Verbesserungen
  • Zugriffe von unerlaubten MAC-Adressen werden auf der Hauptseite angezeigt
  • Komprimierte IMAP-Kommunikation mit Thunderbird 3 verbessert
  • Zeitzone für Firewall-Zeitregeln korrigiert
  • Filtern verschlüsselter Email-Anhänge verbessert
Linux-Basissystem
  • Verbesserung der Sicherheit am Linux-Basissystem (fetchmail-6.3.17: CVE-2010-1167, krb5-1.7.1: CVE-2010-1320, nano-2.0.9: CVE-2010-1160 und CVE-2010-1161, openldap-2.4.19-4, rsync-3.0.7)
  • Aktualisierung von Linux-Basisdiensten (cyrus-imapd-2.3.16, kernel-2.6.32.12, tzdata-2010f, yum-3.2.27)
