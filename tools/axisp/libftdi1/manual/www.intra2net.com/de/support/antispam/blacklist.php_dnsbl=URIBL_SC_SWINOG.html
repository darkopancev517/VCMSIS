Weitere Informationen über SWINOG URIBL

Die Swinog URIBL ist eine Echtzeit-Blacklist die aus Spamfallen zusammengestellt wird. Trifft eine E-Mail in eine solche Spamfalle ein, so werden alle URLs extrahiert, mit einer Whitelist abgeglichen und zur Blackliste hinzugefügt.


Name SWINOG URIBL
Typ URIBL
Website antispam.imp.ch
Lookup uribl.swinog.ch

Spamassassin Einstellungen

urirhsbl        URIBL_SC_SWINOG         uribl.swinog.ch.   A
body            URIBL_SC_SWINOG         eval:check_uridnsbl('URIBL_SC_SWINOG')
describe        URIBL_SC_SWINOG         URI's listed in uribl.swinog.ch.
tflags          URIBL_SC_SWINOG         net
score           URIBL_SC_SWINOG         0 # please adjust the score value
